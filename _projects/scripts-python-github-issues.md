A python script that allows you to delete GitHub issues in bulk. This can be useful for cleaning up old or irrelevant issues from a repository. The script uses the GitHub API and requires a personal access token (classic) with the appropriate permissions to delete issues.

The script is coded to delete closed issues only and can be used as a dry run first. The project is open source and available on GitHub under an MIT licence.
