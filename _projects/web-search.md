A default search engine browser homepage that uses !bangs and shortcodes to route a query to a search engine, or a websites search result page e.g `!amz pepsi max` will show results on amazon uk for my go-to soft drink. 

The app is a simple vanilla HTML, CSS, and JavaScript using AI for the design/layout.
