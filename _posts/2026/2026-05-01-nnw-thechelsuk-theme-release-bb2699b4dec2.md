## thechelsuk

Based on the light and alt themes of my website, adjusted to match the light and darkness of the app.
Theme uses thechelsuk font included in the theme file as well as adapting to the device light/dark mode preference.

## Install this theme in NetNewsWire

Download the `nnwtheme.zip` file and add it manually in NetNewsWire.

[Install now](netnewswire://theme/add?url={{page.download}})
