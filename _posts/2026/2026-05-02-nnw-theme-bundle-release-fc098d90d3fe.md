## Themes

New Retro theme added, for a dark moody look. The guro and thechelsuk themes have been updated to fix some bugs, thechelsuk theme has imbedded JS to handle YouTube feeds by added an imbed player and a link to my favourite video [playing app on iOs](https://extension.app/).

### guro

- Download [guro.zip](https://github.com/thechelsuk/uk.thechels.themes-for-nnw/releases/download/v1.6.2/guro.zip) to install this theme.

### retro

- Download [retro.zip](https://github.com/thechelsuk/uk.thechels.themes-for-nnw/releases/download/v1.6.2/retro.zip) to install this theme.

### thechelsuk

- Download [thechelsuk.zip](https://github.com/thechelsuk/uk.thechels.themes-for-nnw/releases/download/v1.6.2/thechelsuk.zip) to install this theme.
